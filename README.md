# D4 Lord of Hatred — Complete Build Encyclopedia

Encyclopedia-style reference covering ALL 8 classes and every viable build for Diablo 4 Season 13 / Lord of Hatred.

## Coverage
- **All 8 classes** — Barbarian, Warlock, Paladin, Rogue, Sorcerer, Druid, Necromancer, Spiritborn
- **40+ builds** ranked S+ through B with cross-source consensus tier rankings
- **Leveling phases** with skill-by-skill progression for top builds per class
- **Skill bars, gear, stat priorities, talisman sets, runewords, mercenaries** for every build
- **Class mechanics** explained — Arsenal, Soul Shards, Oaths/Arbiter, Specializations, Enchantments, Spirit Boons, Book of the Dead, Spirit Hall
- **Endgame progression checklist** — level 70 through Torment IV

## Features
- Dark gothic Diablo aesthetic with ember particle effects
- Scroll-triggered reveal animations (IntersectionObserver)
- Sticky navigation with class-colored indicators for all 8 classes
- Reading progress bar
- Fully responsive (mobile-friendly)
- Zero dependencies — single HTML file, pure CSS/JS

## Deploy to Vercel

```bash
git init && git add . && git commit -m "init: d4 encyclopedia"
vercel
```

Or connect repo to Vercel dashboard — auto-detects static HTML.
